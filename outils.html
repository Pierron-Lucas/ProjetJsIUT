Outils et organisation

Gantt

Grâce à ce diagramme, cela nous permet de voir notre emploi du temps mise tout au long du projet. Il permet aussi de voir l'avancement du projet

Discord

Ce logiciel de communication, nous permet de communiquer pendant nos heures de projet en créant un serveur pour faciliter le bien dans l'équipe

Github

GitHub permet aux développeurs de modifier, d’adapter et d’améliorer le logiciel gratuitement à partir de référentiels publics, mais facture les interventions pour les référentiels privés, selon trois formules de paiement

VsCode

Visual Studio Code est un éditeur de code open-source développé par Microsoft supportant un très grand nombre de langages grâce à des extensions. Il supporte l’autocomplétion, la coloration syntaxique, le débogage, et les commandes git.
