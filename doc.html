Gantt

Pour créer notre diagramme de Gantt nous avons utilisés le logiciel Notion. Ce dernier nous a permis de planifier correctement notre projet,en effet le diagramme de Gantt permet d'avoir une vue d'ensemble de ce que nous avons accompli, mais aussi de ce qu'il nous reste à faire avant de livrer le projet. Il est donc essentiel de réaliser un Gantt de qualité, pour avoir un projet de bonne qualité.

Discord

En raison de la crise sanitaire qui est actuellement sur le sol français, nous avons décider de réduire les intéractions sociales afin de nous protéger mais aussi de protéger nos proches. C'est pour cela que nous utilisons le logiciel Discord, qui permet de passer des appels qui nous a permis de réaliser l'ensemble de nos réunions mais aussi de pouvoir poser nos différentes questions.

Github

Pour pouvoir nous partager le code et pour avancer sur différentes partie en même temps, nous avons décider d'utiliser le logiciel GitHub qui permet aux différents développeurs de modifier, d’adapter et d’améliorer leurs codes gratuitement

VsCode

Visual Studio Code est un éditeur de code open-source développé par Microsoft supportant un très grand nombre de langages. C'est donc pour cela que nous avons décidé d'utiliser ce dernier afin de mener à bien notre projet.
